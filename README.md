# IGRS Game Classifier Dashboard

## Overview
The IGRS Game Classifier Dashboard is an interactive platform designed to classify games based on various parameters. It provides an intuitive user interface, allowing users to input game details and receive classifications.

## Features
- **User-friendly Interface**: Easily navigate through the dashboard with simple input fields.
- **Game Classification**: Classify games using various algorithms tailored for accuracy.
- **Data Visualization**: Visualize classification results with interactive charts and graphs.
- **Customizable Settings**: Adjust parameters for classification to suit user preferences.

## Getting Started
To get started with the IGRS Game Classifier Dashboard:
1. Clone the repository: `git clone https://github.com/prajnamu/igrs`
2. Install dependencies: run `npm install`
3. Start the application: run `npm start`

## Usage
- Navigate to the dashboard URL provided upon starting the application.
- Input game details in the specified fields.
- Click on the 'Classify' button to receive the classification.
- Review the results displayed on the dashboard, along with visualizations.
